# SafeChain

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

SafeChain is an open community initiative for a wearable anti-chain-snatch safety system.  
The goal is a practical, user-centric, non-injurious design (alert + safe release), with sensor fusion and smart notification.

## Why SafeChain?

- Responds to real-world chain/head jewelry snatch attacks.
- Priority: wearer safety, low false alarms, low power, aesthetically acceptable.
- Includes optional BLE/mobile alert and local haptic/sound alarm.
- Designed to break away safely under extreme pull while notifying user/companions.

## Origin

This project was formed through collaborative innovation, guided by generative AI ideation and analysis (AI-assisted concept and technical discussion).

## What we need

- Electronics architects (MCU/BLE/IMU/strain sensors)
- Embedded firmware engineers (interrupt, low-power, sensors, alarm logic)
- Hardware + mechanical designers (break-away link, compact enclosure, comfort)
- UX/UI + mobile app developers (alert workflow, configuration, pairing)
- Safety researchers & testers (force thresholds, fallbacks, ergonomics)
- Open-source contributors (docs, validation, prototypes, test rigs)

## How to contribute

1. Fork the repo.
2. Add issues for:
   - proposed detection algorithms
   - sensor calibrations
   - safety/certification, power budget
3. Send PRs for:
   - hardware schematics
   - firmware modules
   - test scripts
   - docs and use cases
4. Join discussions (issues / project board / community channels).
